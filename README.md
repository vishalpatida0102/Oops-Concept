# 🧠 Object-Oriented Programming (OOP) Concepts in C++

Welcome to my C++ OOP Concepts repository!  
This repo contains simple, clear, and practical code examples of core object-oriented principles with deep explanation, suitable for:

- Beginners who want to master OOP in C++
- Students preparing for exams/interviews
- Anyone looking to understand real-world implementation of OOP

---

## 📚 Contents Covered

| Concept                    | Description                                                                 |
|---------------------------|-----------------------------------------------------------------------------|
| ✅ Class & Object          | Creating user-defined data types                                            |
| ✅ Constructor             | Default, Parameterized, Copy Constructor                                    |
| ✅ Destructor              | Reverse destruction, memory cleanup                                         |
| ✅ Operator Overloading    | Overloading `+`, `<<`, `[]` and more                                        |
| ✅ Function Overloading    | Compile-time polymorphism                                                   |
| ✅ Function Overriding     | Run-time polymorphism using `virtual` keyword                               |
| ✅ Inheritance             | Single, Multiple, Multilevel, Hybrid                                        |
| ✅ Virtual Functions       | Dynamic dispatch through base class pointers                               |
| ✅ Abstract Classes        | Pure virtual functions and interfaces                                       |
| ✅ Dynamic Memory          | `new` and `delete`, stack vs heap                                           |
| ✅ Polymorphism            | Static & Dynamic                                                            |
| ✅ Encapsulation           | Using private/protected/public access specifiers                           |
| ✅ Friend Function         | Accessing private members from outside                                     |
| ✅ Static Members          | Class-level variables and functions                                         |
| ✅ This Pointer            | Referring to current object                                                 |
